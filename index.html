Para poder comprimir nuestros archivos CSS y JavaScript con Gulp.js realizaremos algunas tareas previamente.

Primero: Abre tu consola de comandos en Windows o tu teminal en MAC.
Segundo: Desde tu terminal accede a la ruta del proyecto.
Tercero: Ejecuta el comando npm install, este comando instalara todos los plugins de la lista devDependencies de package.json
Cuarto: para comprimir tus archivos ejecuta el comando gulp

Listo ahora se creara una carpeta con el nombre de dist con tus archivos comprimidos.
